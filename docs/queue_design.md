# Queue Design

## Overview

Queue is a First-In First-Out (FIFO) data structure.

## Internal Representation

Queue is implemented using a Linked List.

## Operations

- create
- destroy
- enqueue
- dequeue
- front
- size
- is_empty

## Complexity

| Operation | Complexity |
|-----------|------------|
| enqueue   | O(1) |
| dequeue   | O(1) |
| front     | O(1) |
| size      | O(1) |
| is_empty  | O(1) |
