# Linked List Design

## Overview

A singly linked list implementation in C.

## Node

Each node contains:

- data
- next pointer

+------+      +------+      +------+
| 10 | ----> | 20 | ----> | 30 | ----> NULL
+------+      +------+      +------+

## LinkedList

Contains:

- head
- tail
- size

## Operations

- create
- destroy
- push_front
- push_back
- pop_front
- front
- back
- size
- is_empty
