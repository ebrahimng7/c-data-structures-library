# Stack Design

## Overview

Stack is a Last-In First-Out (LIFO) data structure.

## Internal Representation

Stack is implemented using a Linked List.

## Operations

- create
- destroy
- push
- pop
- top
- size
- is_empty

## Complexity

| Operation | Complexity |
|-----------|------------|
| push      | O(1) |
| pop       | O(1) |
| top       | O(1) |
| size      | O(1) |
| is_empty  | O(1) |
